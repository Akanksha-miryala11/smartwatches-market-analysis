# ⌚ Flipkart Smartwatches - Exploratory Data Analysis (EDA)

## 📌 Project Description

This project focuses on performing **Exploratory Data Analysis (EDA)** on a Flipkart Smartwatches dataset using Python. The primary objective is to understand market trends, pricing strategies, customer preferences, and brand performance through statistical analysis and visualization.

The project follows a structured EDA workflow, starting from data cleaning and preprocessing to extracting valuable business insights using various visualization techniques.

---

## 📊 Project Objectives

- Clean and preprocess the dataset
- Perform descriptive statistical analysis
- Analyze smartwatch brands and pricing
- Study customer ratings and reviews
- Understand discount patterns
- Discover relationships among important variables
- Generate business insights through visualizations

---

## 📁 Dataset Information

The dataset contains smartwatch product information available on Flipkart, including:

- Brand
- Product Name
- Selling Price
- Original Price
- Discount Percentage
- Customer Rating
- Number of Ratings/Reviews
- Product Specifications

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/EDA-Flipkart-Smartwatches.git
```

2. Navigate to the project directory

```bash
cd EDA-Flipkart-Smartwatches
```

3. Install the required libraries

```bash
pip install pandas numpy matplotlib seaborn
```

4. Launch Jupyter Notebook

```bash
jupyter notebook
```

5. Open **EDA_Project.ipynb** and run all cells.
